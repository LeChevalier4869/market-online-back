## Admin

### Create product by admin

method: POST
path: /admin/product
authen: true
params: -
body: { name, stock, unit, price, detail, brandId, categoryId }

-------------------------------------------------------------------
### Update product by admin

method: PATCH
path: /admin/product
authen: true
params: productId
body: { name, stock, unit, price, detail, brandId, categoryId }
